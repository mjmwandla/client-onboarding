# OMIG Client Onboarding Portal

Interactive prototype for Old Mutual Investment Group's Segregated Fund client onboarding process.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Deploy to Vercel

```bash
npx vercel
```

## Features

- **28 interactive screens** covering the complete 55-step onboarding journey
- **12 switchable user roles** — each shows only their relevant tasks and navigation
- **Complete FICA/KYC flow** — document upload, verification, pass/fail decision gates
- **Sign-off chain** — interactive multi-team approval pipeline
- **IMA lifecycle** — drafting, review, client signature, mandate register
- **Audit trail** — full compliance logging with expandable detail rows
- **Admin screens** — user management, escalation rules, email templates, deadline configuration

## Tech

React 18 · Vite · Inline Styles (dark institutional theme) · No external dependencies beyond React
